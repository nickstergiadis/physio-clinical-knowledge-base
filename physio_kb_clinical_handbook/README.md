# Physio Clinical Knowledge Base

This repo contains a structured physiotherapy knowledge base and the app that turns it into a searchable, accessible clinical reference.

## Source of truth
All content lives in `/knowledge_base_source`.

## Goal
Build a usable and accessible database-backed web app for clinicians to:
- browse by region and condition
- search all content
- open condition pages with related assessment, rehab, evidence, and postop notes
- preserve citations and source links
- work well on desktop and mobile
