# Physiotherapy Knowledge Base - Clinical Handbook Build

This build cleans up the earlier KB versions and consolidates them into one working clinical-reference package.

## What changed in this version
- normalized the folder structure
- added missing foundations for hip, cervical, elbow/forearm, hand/wrist, and TMJ
- added missing hip, cervical, and hand/wrist assessment / outcome / exercise / evidence pages
- refreshed the master index so it reflects the files actually present
- added a single handbook in both markdown and PDF for quick navigation

## Recommended workflow
1. Open the handbook for an overview.
2. Go to the relevant cluster in the folder tree.
3. Start with the condition page.
4. Open the assessment page beside it.
5. Pull dosage / staging ideas from the rehab progression page.
6. Use post-op annexes only after reconciling them with the actual procedure and restrictions.

## Important notes
- This is a reasoning-support and rehab-reference package.
- It is not a substitute for medical diagnosis, emergency triage, or surgeon-specific orders.
- The evidence update pages are meant to be rechecked periodically as guidelines and systematic reviews evolve.
