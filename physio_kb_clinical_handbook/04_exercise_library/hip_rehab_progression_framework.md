# Hip Rehab Progression Framework

## Phase 1: calm / regain tolerance
- reduce compressive aggravators when lateral hip pain is irritable
- comfortable ROM
- low-load isometrics where helpful
- gait volume adjustment
- trunk and pelvic support work

## Phase 2: rebuild capacity
- hip abductor and extensor strengthening
- rotation and flexion tolerance work as appropriate
- single-leg control
- squat, hinge, step pattern rebuilding
- progressive walking / stair / cycling tolerance

## Phase 3: return to demand
- heavier split squat / deadlift / step work as appropriate
- impact or run progression when relevant
- direction change, pivot, or sport-specific loading
- long-term self-management plan

## Progression principles
- use irritability and 24-hour response to set dosage
- progress load, range, speed, and complexity gradually
- do not chase perfect symmetry before rebuilding function
