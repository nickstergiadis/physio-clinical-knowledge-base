# Ankle Rehab Progression Framework

## Goal
Provide a reusable progression framework for the main non-surgical ankle cluster in this KB:
- lateral ankle sprain
- chronic ankle instability
- midportion Achilles tendinopathy

## General principles
- match exercise stage to irritability and tissue tolerance
- restore function, not just pain scores
- ankle rehab nearly always needs calf capacity plus balance / movement work
- use external support strategically, not as a replacement for rehab
- for tendon pain, modify load rather than stopping all activity

## Phase 1 — Calm + restore access
Use when:
- acute pain / swelling is limiting gait or confidence
- tendon symptoms are irritable and reactive

Targets:
- weight-bearing access
- comfortable motion
- basic calf / foot activation
- clear patient education

Examples:
- pain-limited ROM
- protected weight-bearing progression
- gentle calf activation / double-leg heel raise entry work
- supported balance work
- walking dosage matched to irritability

## Phase 2 — Restore mobility + baseline strength
Use when:
- walking is possible but limited
- pain settles predictably after loading

Targets:
- dorsiflexion restoration
- calf strength / endurance foundation
- static and dynamic single-leg control

Examples:
- dorsiflexion mobility work
- calf raise progressions
- banded inversion / eversion or loaded variants as needed
- step work
- single-leg balance and reach work

## Phase 3 — Progressive capacity
Use when:
- daily function is improving
- recovery after loading is acceptable

Targets:
- multiplanar ankle-foot strength
- sensorimotor control
- tendon load tolerance
- running / landing preparation when relevant

Examples:
- heavier calf loading
- split squat / step-down / loaded carries
- unstable-surface balance or perturbation work
- hopping preparation
- heavy slow resistance or eccentric-concentric tendon loading for Achilles presentations

## Phase 4 — Return to performance / resilience
Use when:
- baseline function is restored enough for higher demand

Targets:
- uneven-ground confidence
- deceleration / landing / cutting mechanics
- plyometric or sport-specific capacity
- recurrence prevention

Examples:
- hops and rebounds
- lateral bounds / cuts / agility drills
- running progression
- reactive balance tasks
- maintenance calf and balance plan

## Condition-specific notes
### Lateral ankle sprain
- early support + progressive loading usually beats prolonged immobilization
- restore dorsiflexion and gait early
- move from swelling management into balance and strength quickly enough

### Chronic ankle instability
- do not stop at band work
- include balance, perturbation, deceleration, and task-specific exposure
- whole-limb and trunk control often matter

### Midportion Achilles tendinopathy
- graded tendon loading is primary treatment
- full rest is usually not indicated in nonacute cases
- calf endurance and later elastic loading matter before full sport return
