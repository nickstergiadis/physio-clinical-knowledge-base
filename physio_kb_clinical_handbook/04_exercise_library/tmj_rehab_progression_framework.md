# TMJ / Orofacial Rehab Progression Framework

## Phase 1: calm pain and reduce overload
### Goals
- reduce constant irritation
- improve comfort with basic jaw function
- reduce parafunctional load

### Common strategies
- education and reassurance
- soft diet temporarily if very irritable
- avoid gum chewing, wide yawns, prolonged clenching
- resting jaw position awareness
- sleep / stress / daytime clenching discussion

## Phase 2: restore comfortable movement
### Goals
- improve pain-limited opening
- normalize movement quality
- reduce guarding

### Common exercise buckets
- controlled opening / closing
- gentle mouth-opening mobility
- lateral deviation / protrusion only when useful
- cervical mobility / postural support work

## Phase 3: improve tolerance and control
### Goals
- build tolerance for chewing / talking / functional jaw use
- reduce recurrence

### Common exercise buckets
- symptom-limited jaw control drills
- graded chewing exposure
- postural and upper-quarter support drills
- self-management plan for flare control

## Criteria themes
- improved comfortable opening
- lower irritability with chewing / talking
- reduced daytime clenching awareness failures
- better self-management of flare-ups

## Dosing principles
- do not aggressively stretch highly irritable joint presentations
- prioritize reversible, conservative care first
- function and pain matter more than eliminating every click
