# Elbow / Forearm Rehab Progression Framework

## Phase 1: calm highly irritable symptoms
### Goals
- reduce flare frequency
- stop repeatedly provoking tendon pain
- maintain enough movement for function

### Common strategies
- task modification
- temporary reduction in gripping / tool / racquet volume
- relative rest, not complete unloading
- pain-guided isometric loading if tolerated
- shoulder / cervical unloading where clearly relevant

## Phase 2: rebuild basic tendon tolerance
### Goals
- restore tolerance to local loading
- improve pain-free grip
- restore wrist / forearm strength

### Common exercise buckets
- wrist extensor loading
- grip loading
- pronation / supination control
- flexor-pronator loading for medial presentations
- scapular / shoulder support work when relevant

## Phase 3: progress capacity
### Goals
- improve endurance and heavier-load tolerance
- close side-to-side functional gap
- restore work / gym / sport confidence

### Common exercise buckets
- heavier slow resistance
- loaded carries
- compound pull / row / landmine / cable patterns as tolerated
- plyometric or speed elements only when needed for sport

## Phase 4: return to task / sport
### Goals
- restore symptom-resilient function
- tolerate real exposure

### Criteria themes
- low irritability at rest and after daily tasks
- improved pain-free grip
- tolerated progressive loading
- task-specific exposure without major next-day flare

## Dosing principles
- use symptom response and next-day flare as the main guide
- avoid chasing pain with too many passive treatments
- expect load capacity to rebuild over weeks, not days
