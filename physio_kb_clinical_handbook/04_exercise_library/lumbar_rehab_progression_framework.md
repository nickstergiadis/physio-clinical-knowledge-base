# Lumbar Rehab Progression Framework

## Purpose
This is a symptom-guided progression framework, not a diagnosis-exclusive program.
Use it for non-specific low back pain and adapt for radicular presentations.

## Stage 1 — Calm and maintain movement
Goals:
- reduce threat and guarding
- maintain activity
- find tolerable movement windows

Targets:
- short walking bouts
- repeated low-irritability lumbar movement if helpful
- breathing / bracing awareness without over-rigidity
- gentle trunk / hip activation
- frequent position changes

Examples:
- unloaded flexion or extension bias only if clearly helpful
- pelvic tilts / lower trunk movement
- hooklying march / brace drill
- bridge regressions
- sit-to-stand from tolerated height
- short frequent walks

## Stage 2 — Restore function
Goals:
- improve tolerance to daily tasks
- regain confidence with movement
- reduce flare frequency

Targets:
- trunk endurance
- hip extensor / abductor strength
- squat / hinge patterning
- step and carry tolerance
- sitting and standing capacity

Examples:
- bridge progressions
- dead bug / bird dog regressions and progressions
- hip hinge drills
- goblet squat to box / sit-to-stand progression
- split squat / step-up progressions
- loaded carries
- walking volume progression

## Stage 3 — Build capacity
Goals:
- improve resilience to work, gym, and lifestyle demands
- reduce recurrence risk

Targets:
- progressive resistance training
- whole-body conditioning
- repeated-load tolerance
- work / sport-specific exposure

Examples:
- RDL / hinge progression
- squat progression
- row / pull / push integration
- single-leg work
- trunk endurance circuits
- aerobic conditioning
- graded lifting and return-to-gym exposure

## Radicular pain modifications
- prioritize symptom behaviour in the leg over back soreness
- avoid aggressive neural provocation in high-irritability stages
- progress neural loading gradually only when appropriate
- monitor neurological change, not just pain intensity

## Programming principles
- dose to irritability
- avoid all-or-nothing cycles
- progress one variable at a time: range, load, reps, tempo, exposure, complexity
- use flare plans rather than stopping all activity
- keep education and self-management integrated with loading
