# Knee Rehab Progression Framework

## Goal
Provide a reusable framework for progressing most non-surgical knee presentations in this KB:
- knee OA
- patellofemoral pain
- degenerative meniscal presentation

## General principles
- start from the dominant irritable task
- modify load, not all activity
- keep a meaningful functional anchor
- progress based on symptom response over 24 hours, not fear alone
- build quadriceps capacity in almost every knee pathway
- include hip, calf, and whole-limb loading rather than isolating the knee forever

## Phase 1 — Calm + maintain
Use when:
- pain is irritable
- flare is recent
- swelling or reactivity is limiting confidence

Targets:
- maintain motion, especially extension
- reduce threat / fear
- keep walking or cycling dose tolerable
- begin easy strength work

Examples:
- heel slides / extension positioning as needed
- sit-to-stand from higher surface
- short-arc or tolerable knee extensor work
- bridge / calf raise / simple hip work
- step count or walking dose matched to irritability

## Phase 2 — Restore capacity
Use when:
- baseline daily function is possible but limited
- symptoms recover acceptably after loading

Targets:
- quadriceps strength
- hip strength
- controlled squat / stair tolerance
- confidence in single-leg loading

Examples:
- sit-to-stand progression
- squat to box
- split squat
- step-up / controlled step-down
- leg press or equivalent
- calf raises
- single-leg balance
- bike / walking progression

## Phase 3 — Higher-load function
Use when:
- basic ADLs are improving
- recovery after loading is acceptable

Targets:
- deeper flexion tolerance where relevant
- heavier strength work
- stairs, hills, kneeling, transfers, hiking, gym tasks

Examples:
- deeper squat progression
- lunge variations
- split squat / rear-foot elevated split squat as tolerated
- heavier leg press / machine or free-weight loading
- carries / sleds / incline walking when useful
- rotational tolerance drills if relevant for meniscal or sport demands

## Phase 4 — Return to performance / resilience
Use when:
- symptoms are controlled enough for higher-demand goals

Targets:
- return to running / sport / work demands
- tolerance for unpredictable loading
- flare self-management
- long-term maintenance

Examples:
- plyometric / hop / deceleration work if needed
- running reintroduction
- agility / sport drills
- occupational kneeling / lifting / ladder exposure
- strength maintenance plan

## Condition-specific notes
### Knee OA
- expect some discomfort with loading
- emphasize consistency over novelty
- pair strength with aerobic work
- use flare planning rather than stopping rehab completely

### Patellofemoral pain
- track response to stairs, squat depth, and running
- use knee-targeted and often hip-targeted strengthening
- consider taping / orthoses / retraining only if they help progression

### Degenerative meniscal presentation
- restore extension early if limited
- progress loaded flexion and rotation gradually
- do not assume clicking alone means failure of rehab
