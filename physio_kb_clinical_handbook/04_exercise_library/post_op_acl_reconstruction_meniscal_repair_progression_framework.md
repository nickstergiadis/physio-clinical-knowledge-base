# Post-Op ACL Reconstruction + Meniscal Repair Progression Framework

## Phase 1: protection / activation
- pain and effusion control
- full passive extension target
- flexion progression per protocol
- quadriceps activation
- safe transfers, stairs, and brace use
- WB and gait according to surgeon instructions

## Phase 2: controlled strength + gait
- progressive closed-chain loading
- terminal knee extension control
- hip / trunk strength
- bike and conditioning as tolerated / allowed
- balance and single-leg control once appropriate

## Phase 3: capacity rebuilding
- split squat / squat / hinge progressions
- step-down and deceleration control
- single-leg strength
- hop-prep and landing mechanics
- running preparation when criteria are met

## Phase 4: return to running / sport
- criterion-based running entry
- hopping, plyometrics, COD
- sport-position drills
- confidence / psychological readiness
- return-to-sport testing battery

## Suggested progression markers
- quiet knee / manageable effusion
- full passive extension
- flexion appropriate to phase
- no extensor lag
- acceptable gait
- strength and hop criteria as phase advances
