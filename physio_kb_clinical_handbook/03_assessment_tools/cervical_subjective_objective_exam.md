# Cervical Subjective + Objective Exam

## Subjective priorities
- trauma vs gradual onset
- headache pattern and arm symptoms
- dizziness, visual change, facial symptoms, swallowing / speech issues when indicated
- sleep, work posture, driving, lifting, gym exposure
- irritability and 24-hour pattern
- beliefs about discs, posture, and “pinched nerves”

## Objective priorities
- cervical ROM with symptom response
- repeated movement / sustained posture only if useful
- upper-quarter neuro screen when arm symptoms present
- cranio-cervical flexion / endurance or similar low-load control testing
- thoracic and shoulder-girdle contribution
- cervicogenic headache support tests when clinically indicated

## Documentation essentials
- neck-dominant vs arm-dominant vs headache-dominant
- irritability
- neuro screen findings
- comparable task / movement
