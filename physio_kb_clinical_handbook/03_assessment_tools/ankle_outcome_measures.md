# Ankle Outcome Measures

## Core ankle tools in this KB
### Lateral ankle sprain / chronic ankle instability
- LEFS
- FAAM / FAAM-Sport when available in your workflow
- CAIT if perceived instability is a major feature
- modified Star Excursion Balance Test (mSEBT / Y-balance style use if appropriate)

### Midportion Achilles tendinopathy
- VISA-A
- LEFS or FAAM for broader lower-limb activity context
- single-leg calf raise endurance
- hop / spring tasks if relevant to goals and stage

## Practical use notes
### LEFS
Useful for:
- broad lower-limb function
- consistent outpatient follow-up
- sprain / instability and Achilles clusters

### FAAM / FAAM-Sport
Useful for:
- foot/ankle-specific function
- athletic or return-to-sport tracking

### CAIT
Useful for:
- capturing perceived instability severity
- recurrent “giving way” presentations

### mSEBT / dynamic balance tests
Useful for:
- CAI and post-sprain progression
- identifying side-to-side balance / reach deficits
- return-to-sport style monitoring when appropriate

### VISA-A
Useful for:
- Achilles-specific symptoms and function
- monitoring tendon-focused rehab over time

## Practical measurement domains to pair with questionnaires
- weight-bearing dorsiflexion
- single-leg balance
- single-leg calf raise count / endurance
- hop tolerance where relevant
- walking / running tolerance
- confidence / fear around sport or uneven-ground loading
