# Hip Subjective + Objective Exam

## Subjective priorities
- pain location: groin, lateral hip, buttock, anterior thigh
- onset: gradual vs traumatic vs post-op
- aggravating tasks: sitting, stairs, side-lying, walking, deep squat, pivoting, running
- morning stiffness / startup pain
- night pain and sleep position
- lumbar symptoms or neurologic features
- sport, gym, running, and work demands

## Objective priorities
### Baseline movement
- gait
- sit-to-stand
- squat / hinge
- single-leg stance
- step-up / step-down

### ROM
- flexion
- extension
- internal rotation
- external rotation
- compare symptom response and end-feel pattern

### Strength / control
- abductors
- external rotators
- extensors
- trunk / pelvic control in functional tasks

### Provocation cluster
- FADIR and related flexion-loading tests when FAIS/intra-articular pattern suspected
- FABER and quadrant-style tests as pattern support only
- lateral hip palpation and single-leg loading for GTPS pattern support

### Adjacent region screen
- lumbar screen if indicated
- adductor / abdominal contribution when relevant

## Documentation essentials
- dominant pain region
- irritability
- comparable sign / task
- ROM pattern
- load tolerance in single-leg and functional tasks
