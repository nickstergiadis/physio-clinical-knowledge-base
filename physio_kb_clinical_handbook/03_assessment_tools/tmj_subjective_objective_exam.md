# TMJ / Orofacial Subjective + Objective Exam

## Subjective
### Key questions
- pain location: joint-line, cheek, temple, ear, teeth, diffuse
- pain vs clicking vs locking vs opening restriction
- chewing, yawning, talking, singing, dental work tolerance
- clenching / bruxism / gum chewing
- morning stiffness or fatigue
- headache and cervical symptoms
- trauma or wide-yawn / dental procedure onset
- prior splint / dental / physio response

## Objective
### Observation
- resting jaw position
- opening path
- guarding, asymmetry, obvious clench pattern

### Range / function
- comfortable mouth opening
- maximum mouth opening
- lateral deviation and protrusion
- chewing simulation or symptom-relevant functional task if appropriate

### Provocation
- clench / bite provocation
- palpation of masseter / temporalis and symptomatic tissues
- joint-line sensitivity
- note clicking / popping / catch only if clinically relevant

### Associated assessment
- cervical AROM and symptom relationship
- posture / upper-quarter tension where relevant

## High-yield interpretations
- muscle tenderness + painful chewing / clenching → myogenous TMD likely
- clicking with decent function and low pain → often monitor / educate rather than over-treat
- marked opening restriction / lock history → mobility-dominant disc / joint presentation
