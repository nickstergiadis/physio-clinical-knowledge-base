# Lumbar Subjective + Objective Exam

## Subjective screen
### Core symptom questions
- Where is the pain?
- Is it back-dominant, buttock-dominant, or leg-dominant?
- Does it go below the knee?
- Any numbness, tingling, weakness, giving way?
- What makes it worse / better?
- What is the 24-hour pattern?
- How irritable is it and how long do flares last?

### Functional impact
- sitting tolerance
- standing tolerance
- walking tolerance
- transfers / bed mobility
- bending / dressing / lifting
- work and driving tolerance
- sleep disruption
- exercise / gym / sport effect

### Red-flag screen
Ask about:
- trauma
- cancer history / unexplained weight loss
- fever / infection risk
- steroid use / osteoporosis
- saddle numbness
- bladder / bowel change
- rapidly worsening weakness
- inflammatory-type pattern if relevant

### Psychosocial / prognostic screen
- fear of movement / damage beliefs
- distress, low mood, high threat beliefs
- work concerns / compensation context where relevant
- recovery expectations
- previous care and its effect
- consider STarT Back or equivalent risk stratification tool

## Objective screen
### Observation
- posture only as context, not diagnosis
- transfer behaviour
- gait quality
- guarded behaviour / movement confidence

### Baseline movement
- flexion
- extension
- side bend
- repeated movement or sustained loading only if clinically useful
- symptom response: centralization, peripheralization, no effect, increased local pain

### Functional tests
- sit-to-stand
- squat / hinge
- step-up
- heel / toe walk if radicular features suspected
- loaded carry or work-specific pattern if appropriate

### Neurological screen (when indicated)
- myotomes
- dermatomes
- reflexes
- upper motor neuron screen if indicated by history / presentation

### Neural provocation (when indicated)
- slump / SLR or alternatives based on irritability and purpose
- interpret as part of the whole pattern, not in isolation

### Adjacent region screen
- hip ROM and provocative movements
- SIJ / buttock region as clinically relevant
- vascular / peripheral neural contributors if indicated

## High-value documentation points
- dominant classification: axial LBP vs radicular pain vs mixed
- irritability
- key functional limits
- red-flag findings or absence thereof
- baseline comparable sign / task
- clear plan and progression target
