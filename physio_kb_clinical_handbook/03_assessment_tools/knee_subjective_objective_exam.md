# Knee Subjective + Objective Exam

## Subjective priorities
### History of present condition
- exact location: anterior, medial joint line, lateral joint line, diffuse, posterior
- onset: traumatic, overload, gradual, flare of known OA
- aggravators: stairs, squat, sit-to-stand, kneeling, running, hills, twisting, prolonged sitting
- easing factors
- irritability and 24-hour pattern
- swelling: immediate, delayed, absent
- locking: true block versus painful click/catch
- instability: giving way versus confidence issue
- prior imaging / injections / surgery
- work, sport, and exercise exposures
- patient understanding / expectations

### Red flags / urgent considerations
- inability to weight-bear after trauma
- large rapid effusion after trauma
- suspected fracture
- hot swollen unwell joint / infection concern
- DVT or vascular concern
- significant calf symptoms not explained by the knee alone
- true locked knee / major extension block

## Objective priorities
### Observation
- gait
- swelling / effusion
- quadriceps bulk
- alignment / stance
- willingness to load

### Range of motion
- active flexion / extension
- passive overpressure if appropriate
- note extension loss carefully

### Functional testing
- sit-to-stand
- squat
- step-up / step-down
- single-leg stance / single-leg squat if appropriate
- gait and stair observation

### Strength / capacity
- quadriceps
- hamstrings
- calf
- hip abductors / extensors
- compare pain response and control quality

### Region-specific exam emphasis
#### OA-leaning
- range loss
- functional tolerance
- quadriceps weakness
- activity-related pain / stiffness pattern

#### PFP-leaning
- pain with squat or step-down
- anterior knee pain provocation
- knee extensor loading
- hip / trunk contribution
- running / jumping tolerance when relevant

#### Meniscal-leaning
- joint-line symptoms
- loaded flexion / rotation behavior
- extension block
- effusion
- use meniscal tests as part of the whole picture, not in isolation

### Optional special tests
Use when they meaningfully shift probability:
- patellofemoral provocation / compression-related testing
- McMurray
- Thessaly (only if appropriate / tolerated)
- ligament testing if trauma or instability history warrants it

## What to document clearly
- baseline pain-provoking task
- extension status
- key function benchmark (e.g. stair descent, squat depth, 30-second sit-to-stand)
- irritability after testing
- patient goals

## Suggested quick summary line
“Clinical picture most consistent with [OA / PFP / degenerative meniscal presentation], with key contributors including [load tolerance / strength / hip control / ROM / irritability], no current red flags, plan for active management and reassessment.”
