# Shoulder Subjective + Objective Exam

## Subjective exam priorities
### Triage first
- fracture / unreduced dislocation suspicion
- acute traumatic inability to elevate
- infection / tumor / red flag pattern
- neurovascular symptoms
- cervical referral or neurological suspicion

### Clarify the dominant presentation
Ask whether the problem feels mainly like:
- pain with loading
- stiffness / movement loss
- instability / slipping / apprehension

### High-yield subjective questions
- exact location and spread of symptoms
- traumatic vs gradual onset
- overhead, away-from-body, sleep, dressing, and hand-behind-back aggravators
- prior dislocations or reduction
- diabetes / thyroid history
- collision / contact / overhead sport demands
- prior imaging, injection, or surgery
- patient goals and timelines

## Objective exam flow
### 1. Observation
- protective posture
- swelling / deformity / bruising
- resting scapular position
- obvious AC step deformity or post-dislocation signs

### 2. Active ROM
- flexion
- abduction / scaption
- external rotation
- hand-behind-head / hand-behind-back
Document:
- painful arc?
- end-range stiffness?
- apprehension?

### 3. Passive ROM
- external rotation at side
- flexion
- abduction
- internal rotation in a clinically meaningful position
Prioritize pattern recognition:
- marked passive restriction, especially external rotation → think frozen shoulder
- near-preserved passive ROM with painful active / resisted loading → think RCRSP
- guarded outer-range symptoms after trauma → think instability

### 4. Resisted testing
- external rotation
- abduction / scaption
- internal rotation
- elbow flexion if biceps involvement is relevant
Use as symptom provocation + functional capacity assessment, not purely tissue labeling.

### 5. Instability testing when appropriate
Only once clinically appropriate:
- apprehension
- relocation / release concept
- load-and-shift / laxity context where skilled and indicated
Document confidence / fear response, not only pain.

### 6. Scapular / kinetic-chain assessment
- scapular assistance or cue response
- thoracic extension / rotation contribution
- task-specific movement strategy
- whole-chain contribution in athletes

### 7. Cervical / neurological screen when indicated
- cervical ROM / symptom behavior
- neuro screen if symptoms suggest neural involvement
- consider referred pain when local shoulder findings are weak

## Red flags / escalation triggers
- fracture suspicion
- unreduced dislocation / acute instability event
- major traumatic cuff tear suspicion
- infection
- tumor suspicion / non-mechanical constant pain
- progressive neurological signs
- vascular compromise

## Differential diagnosis prompts
### More likely RCRSP
- load-related shoulder pain
- painful elevation
- resisted loading sensitivity
- passive ROM near preserved

### More likely adhesive capsulitis
- global passive stiffness
- external rotation loss marked
- slow onset and prolonged restriction
- diabetes / thyroid association

### More likely traumatic anterior instability
- trauma history
- apprehension / slipping
- recurrent episodes / high-risk sport
- age and activity profile raise recurrence concern

## Optional outcome-linked baselines
- pain with a meaningful aggravating task
- active flexion / abduction
- passive external rotation
- QuickDASH or SPADI
- instability confidence / return-to-sport concerns where relevant
