# Ankle Subjective + Objective Exam

## Subjective priorities
### History of present condition
- exact location: lateral ligaments, high ankle, posterior heel, insertional heel, diffuse ankle, medial ankle
- onset: inversion injury, gradual overload, recurrent sprain history, training spike
- ability to weight-bear immediately and currently
- swelling timing and bruising
- giving way / recurrent sprain history
- locking / catching / deep joint pain
- aggravators: walking, stairs, uneven ground, running, cutting, hopping, hills, calf raises
- easing factors
- irritability and 24-hour pattern
- prior imaging, bracing, tape, boots, injections, or surgery
- sport / work / footwear context

### Red flags / urgent considerations
- inability to weight-bear with fracture concern
- Ottawa-rule-positive features
- suspected syndesmosis injury
- Achilles rupture concern (pop, major push-off weakness)
- hot, swollen, systemically unwell patient / infection concern
- calf pain/swelling raising DVT concern
- neurovascular compromise

## Objective priorities
### Observation
- gait and stance tolerance
- swelling / bruising distribution
- deformity or major asymmetry
- willingness to load

### Range of motion
- active and passive plantarflexion / dorsiflexion
- inversion / eversion where appropriate
- weight-bearing dorsiflexion / lunge-type testing when relevant

### Palpation
- lateral ligament region
- distal fibula / malleoli / base of fifth / navicular when fracture screening matters
- syndesmosis region if mechanism suggests
- Achilles midportion vs insertion

### Functional testing
- weight-bearing tolerance
- double-leg and single-leg heel raise
- single-leg balance
- squat / step-down
- hop / landing tests when stage and irritability allow
- running / deceleration screen when clinically relevant

### Region-specific emphasis
#### Acute lateral ankle sprain
- fracture screening first
- swelling / pain distribution
- dorsiflexion loss
- gait and weight-bearing progression

#### Chronic ankle instability
- dynamic balance
- single-leg control
- calf / peroneal strength
- hop / landing confidence
- movement quality under perturbation or sport-like demand

#### Midportion Achilles tendinopathy
- exact symptom location
- calf raise pain and endurance
- tendon loading response
- dorsiflexion and subtalar mobility where relevant
- running / hop reactivity if appropriate

## Outcome-focused benchmarks to document
- walking tolerance
- stair tolerance
- single-leg balance time / quality
- weight-bearing dorsiflexion measure
- single-leg calf raise count / pain response
- hop or running benchmark if relevant to goals

## Quick summary line
“Presentation most consistent with [acute LAS / CAI / midportion Achilles tendinopathy], currently limited by [weight-bearing tolerance / dorsiflexion loss / instability / calf capacity / tendon-loading pain], with [no current red flags / further imaging review indicated].”
