# Elbow / Forearm Subjective + Objective Exam

## Subjective
### Key questions
- exact symptom location
- gripping / lifting / tools / keyboard / mouse / sport aggravators
- pronation / supination irritability
- trauma vs gradual onset
- neural symptoms
- cervical / shoulder symptoms
- dominant hand and occupational load
- previous injection / brace / rehab response

## Objective
### Observation
- resting posture of elbow / wrist
- swelling, atrophy, protective posture
- obvious carrying-angle asymmetry if relevant

### Active motion
- elbow flexion / extension
- pronation / supination
- wrist motion
- symptom behaviour with repeated gripping / wrist extension / pronation tasks

### Resisted tests
- wrist extension
- middle finger extension if useful
- gripping / pain-free grip
- wrist flexion / pronation
- biceps / triceps as relevant

### Palpation
- lateral epicondyle / common extensor origin
- medial epicondyle / flexor-pronator origin
- proximal radial tunnel region when indicated
- ulnar nerve region / cubital tunnel when indicated

### Screening
- cervical screen if symptoms are atypical / persistent
- shoulder screen where kinetic-chain drivers are likely
- neuro screen if tingling, numbness, or weakness reported

## High-yield interpretations
- lateral epicondyle pain + painful grip / resisted wrist extension → likely LET
- medial epicondyle pain + painful pronation / grip → likely medial elbow tendon overload
- proximal forearm ache with neural features → consider radial tunnel contribution
- ring / little finger sensory symptoms → consider ulnar nerve contribution
