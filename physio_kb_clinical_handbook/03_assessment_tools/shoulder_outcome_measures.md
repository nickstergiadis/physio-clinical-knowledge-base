# Shoulder Outcome Measures

## Goal
Use a small set of repeatable measures that match the dominant presentation and are practical in outpatient practice.

## Core generic shoulder tools
### SPADI
Useful for:
- rotator cuff-related shoulder pain
- adhesive capsulitis
Strengths:
- pain + disability domains
- widely used in shoulder rehab literature

### QuickDASH / DASH
Useful for:
- broader upper-limb function
- work and daily task impact
Strengths:
- captures function beyond isolated shoulder range

## Presentation-specific metrics
### For rotator cuff-related shoulder pain
- painful task rating (reach, press, lift, sleep, overhead)
- active elevation tolerance
- resisted loading tolerance
- SPADI or QuickDASH

### For adhesive capsulitis
- passive external rotation
- active hand-behind-back / overhead access
- SPADI
- sleep disturbance / dressing difficulty rating

### For anterior shoulder instability
- recurrence episodes
- apprehension during sport-specific position
- confidence / readiness for return to sport
- instability-specific scales where relevant to setting and population

## Performance / return-to-sport measures
Depending on population:
- closed-chain upper-extremity stability style tests
- push / press / carry tolerance
- plyometric or throwing exposure markers
- criteria-based return-to-sport battery for instability pathways

## Practical recommendation
For most outpatient shoulder patients, track:
- one symptom task
- one ROM metric
- one patient-reported outcome measure
- one strength / capacity marker

## Notes
- do not overload the chart with too many scales
- choose measures that you will actually repeat
- pair PROMs with functional task testing
