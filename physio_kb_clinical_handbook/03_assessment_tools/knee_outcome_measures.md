# Knee Outcome Measures

## Core options
### KOOS / KOOS-JR
Useful for:
- knee OA
- post-op / broader knee symptom tracking
Comments:
- KOOS-JR is quicker and pragmatic for routine follow-up
- full KOOS gives more domain detail

### WOMAC
Useful for:
- knee OA pain, stiffness, function
Comments:
- strong OA relevance
- useful when you want OA-specific change tracking

### AKPS / Kujala score
Useful for:
- patellofemoral pain
Comments:
- practical when the dominant syndrome is anterior knee pain / PFP

### LEFS
Useful for:
- broad lower-limb function across diagnoses
Comments:
- fast, simple, and useful when you want a cross-condition tool

### Numeric Pain Rating Scale
Useful for:
- current pain
- aggravating task pain
Comments:
- pair with a function marker, not pain alone

### Patient-Specific Functional Scale
Useful for:
- individualized goals such as stairs, kneeling, running, golf, hiking
Comments:
- excellent for meaningful patient-centered tracking

### 30-second sit-to-stand / timed stair / walk tolerance
Useful for:
- knee OA and general functional capacity
Comments:
- objective functional anchors often communicate progress better than pain scores alone

## Suggested pairing by condition
### Knee OA
- KOOS-JR or WOMAC
- NPRS
- PSFS
- sit-to-stand / walking / stair benchmark

### Patellofemoral pain
- AKPS/Kujala
- NPRS during squat / stairs / run
- PSFS
- functional loading benchmark

### Degenerative meniscal presentation
- LEFS or KOOS-JR
- NPRS
- PSFS
- squat / stair / walking benchmark

## Practical rule
Use:
- one region-specific PROM
- one pain score
- one patient-specific function measure
- one repeatable performance task

That is usually enough for busy clinical follow-up.
