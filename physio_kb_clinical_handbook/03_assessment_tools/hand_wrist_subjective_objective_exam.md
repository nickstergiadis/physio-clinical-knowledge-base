# Hand / Wrist Subjective + Objective Exam

## Subjective priorities
- exact symptom map: radial styloid, dorsal wrist, volar wrist, thumb base, ulnar wrist
- childcare, lifting, keyboard / mouse, phone, sport, and gym triggers
- clicking, locking, numbness, night symptoms
- post-partum or recent load spike context when relevant
- previous splint / injection / imaging history

## Objective priorities
- AROM and symptom response
- thumb and wrist resisted tests
- grip / pinch tolerance if relevant
- first dorsal compartment provocation tests interpreted cautiously
- screen thumb CMC and neural contributors when indicated
- proximal forearm / elbow contribution when relevant

## Documentation essentials
- exact pain region
- irritability
- key task trigger
- comparable grip, pinch, or lifting task
