# Condition Page Template (Quick View + Deep View)

Reusable clinician-facing structure for all condition pages.

## Quick View
- Definition / clinical framing
- Common presentation
- Key differentials
- Exam priorities
- Red-flag screen
- First-line management
- Prognosis / timeline snapshot

## Deep View
- Differential reasoning
- High-yield tests and interpretation
- Subgrouping / modifiers
- Treatment options
- Exercise / progression strategy
- Follow-up markers / reassessment
- Evidence summary
- Linked related pages

## Citation standard
- Use markdown links for references when available.
- Use clearly marked placeholders only when evidence details are still pending (e.g., `PLACEHOLDER CITATION - verify source`).
