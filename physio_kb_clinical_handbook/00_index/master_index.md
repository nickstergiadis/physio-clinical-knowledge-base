# Physiotherapy Knowledge Base - Clinical Handbook Build

This is the cleaned and consolidated handbook build of the physiotherapy knowledge base. It merges the earlier modular clusters and case-based additions, then normalizes the structure so each major cluster has:
- foundations
- condition pages
- assessment tools
- exercise progression support
- evidence update pages
- post-op annexes where relevant

## Navigation
- Start with the **condition** page that best matches the presentation.
- Pair it with the relevant **assessment tool**.
- Use the **exercise framework** for stage-based progression logic.
- Use **protocol annexes** only after checking the operative note and surgeon-specific restrictions.
- Review the **evidence update** page when management assumptions may have shifted.
- Build new condition pages with the [Condition Page Template (Quick View + Deep View)](./condition_page_template.md).
