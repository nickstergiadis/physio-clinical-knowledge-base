# Hip Arthroscopy for FAI / Labral Repair Post-Op Notes

## Use this annex for
- hip arthroscopy after cam / pincer correction
- labral repair or debridement
- quick rehab framing before reconciling with surgeon restrictions

## Non-negotiable note
Always check:
- exact procedure performed
- weight-bearing status
- ROM restrictions (especially flexion, external rotation, extension)
- whether microfracture / cartilage procedure was added
- capsular repair status
- surgeon timeline for bike, strengthening, running, and sport

## Practical rehab frame
### Early protection phase
- protect surgical tissues and respect WB / ROM restrictions
- pain and swelling control
- gait normalization within allowed loading
- gentle ROM only as allowed
- basic trunk, glute, and core activation without provoking hip irritability

### Motion and control phase
- restore hip ROM gradually
- improve lumbopelvic control
- progress to low-load closed-chain work as allowed
- normalize daily function and stair tolerance

### Strength phase
- progressive hip abductor / extensor / rotator strengthening
- single-leg control work
- progressive machine and functional strengthening when tolerated
- avoid rushing compressive / pivoting load if symptoms remain reactive

### Return-to-running / field-court phase
- running progression only when pain, ROM, strength, and single-leg control criteria are acceptable
- progress change of direction, acceleration-deceleration, and sport-specific work later
- criteria-based return is preferable to time alone

## Common pitfalls
- pushing end-range flexion or rotational load too early
- neglecting gait and pelvic control
- underdosing strength once early pain settles
- ignoring irritability after sitting, squatting, or pivoting progression

## Evidence update note
Post-arthroscopy outcomes for FAI are generally favorable in appropriately selected patients, but procedure details and surgeon restrictions materially change rehab. Use milestones and tissue context, not generic time-based progression alone.
