# AGENTS.md

## Project goal
Build a usable and accessible clinician-facing physiotherapy knowledge-base app from the markdown source in `/knowledge_base_source`.

## Source of truth
Do not rewrite the project into a CMS-first workflow.
Markdown files in `/knowledge_base_source` are the source of truth.

## Priorities
1. Accessibility
2. Reliability
3. Searchability
4. Clear clinical navigation
5. Clean mobile layout
6. Maintainability

## Constraints
- Keep the UI minimal and clinical.
- Preserve citations and evidence notes.
- Prefer boring, stable architecture over cleverness.
- Use TypeScript.
- Use clear file and component names.
- Keep pages print-friendly.
- Do not remove source content.
- Do not add AI/chat features until core browse and search are solid.

## Workflow
- Inspect actual repo content before coding.
- Make focused changes.
- Run checks after changes.
- Update README when setup or workflow changes.
